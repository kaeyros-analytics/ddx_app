The DDX IoT Dynamic Pricing App is designed to support Risk Managers in the pricing setting process. It can be used to gather real-time information about internal and external factors that can have impact on the premium rate and ultimately enable Risk Manager to determine for a given client and a given machine type the optimal premium rate.

The Application uses the PAO (Predict-Optimize) + Forecast Framework. The margin rate will be predicted in a first step as a function of various parameters such as IoT_Health_Index, competitor price, consumer confidence index, geopolitical risk index, weather, demand for industrial good, inflation rate among others.

The Back end of the App contains a data pipeline and for data preparation/transformation for modelling purposes, a PAO-Framework using Bayes Hierarchical regression Model, a Linear Programming model for Optimization (Thompson) and a Forecast-Engine (VAR, VARMA) to forecast future premium rate s in the next 3,6, 12 months.

The Front end contains a main tab for forecasting activities and sub-tabs showing results of the Explorative Data Analysis using predictors and target variable.
